Projects and case studies

Penetration testing: E‑commerce platform

Web App

Discovered SQLi and XSS vectors through authenticated testing. Partnered with engineering to implement parameterized queries, output encoding, and CSP. Risk exposure reduced by ~80% across critical paths.

  • Scope: Auth flows, payment, admin
  • Tech: Burp Suite, custom scripts
  • Outcome: Hardened inputs, reduced exploitability

Incident response: Financial services drill

IR

Ran tabletop + live detection exercise. Built runbooks for triage, containment, and recovery. MTTR dropped from 72h to 24h with post‑incident lessons integrated into SIEM detections.

  • Scope: Endpoint compromise, lateral movement
  • Tech: Splunk, EDR
  • Outcome: Faster recovery, clearer comms

Secure web development: Auth hardening

AppSec

Implemented MFA, session rotation, and secure cookie flags; audited password storage and crypto usage. Achieved alignment with OWASP ASVS controls for authentication and session management.

  • Scope: Login, sessions, account recovery
  • Tech: Backend + CDN security features
  • Outcome: Reduced account takeover risk
